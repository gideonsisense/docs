Attaching and Detaching Directories

You can change the physical location of the ’s data directory by attaching and detaching the directory. Detaching an unregisters it from the server, but not delete the data directory.

Detaching an

  1. From the Start menu, open the Server Console.
  2. Click on the relevant to view the menu panel.
  3. Click Detach to detach the relevant .

Attaching an

  1. From the Start menu, open the Server Console.
  2. In the top panel of the console, click the Attach folder icon .
  3. Select the location of the folder and click OK.
