Field Suggestions

To quickly and easily create meaningful visualizations in your dashboard, displays field suggestions to Dashboard Designers when creating new widgets.

After creating your first dashboard, begins to display field suggestions the next time you create a widget.

When determining what fields to suggest, looks at fields being used in other widgets on the same server, which fields are typically working together in other widgets, and what fields you have already selected.

Fields that are already being used in the widget are not suggested.

has exposed the POST /suggestions API endpoint in the 1.0 REST API that enables you to retrieve a list of suggestions based on ’s algorithm for calculating field suggestions. This is useful for displaying suggested fields in your own application. For more information, see the e REST API documentation.
