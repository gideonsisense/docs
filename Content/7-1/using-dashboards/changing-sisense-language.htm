Changing ’s Language

This topic discusses how Sisense Administrators can change the language of the Sisense Web Application for all of their users.

Administrators can set the default language for a server from the Admin page. If you are an Administrator and want to set the language for your system, users groups, or users, see Changing the ’s Language.

If your users would like to set another language for the other than the language defined by the Administrator, theyyou can select a language from theyour User Profiles or from the Login page.

supports the following languages:

  • English
  • Chinese (Simple)
  • Dutch
  • French (France)
  • German
  • Japanese
  • Portuguese (BR)
  • Russian
  • Spanish (ES)
  • Spanish (LA)
  • Italian

In addition, any languages that your company have translated will appear in the list according to the name defined in the translated system files.

Note, the following content is not being translated:

  • Built-in Javascript editor
  • and Server Console
  • documentation and online help
  • REST API and documentation
  • automated emails
