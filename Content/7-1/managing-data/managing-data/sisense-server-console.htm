Changing the Location of the Data Storage

To change the folder:

  1. From the Windows start menu, open the Server Console.
  2. In the top panel, click on the cog icon to view/change server preferences.
  3. Click Browse next to Default server data folder.
  4. Select the location of the folder where you want to store data, and click OK.

Canceling Running Queries

You can cancel all running queries in your to free up resources. This is useful in cases where your is consuming too many resources, and as a result, causing the to be unresponsive.

To cancel all queries:

  1. Click on the Windows start menu, and select All Programs > > Server Console.
  2. Click the relevant to view the menu panel.
  3. Click Cancel All Queries.
