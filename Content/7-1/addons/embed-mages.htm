Embed Images in Pivot Tables

The Embed Images add-on allows you to insert images into pivot or table widget cells.

Note:

  • The image URL must be defined within the ElastiCube for this add-on to work. If your database does not have a field with this information, it can usually be created by using a Custom Field in your ElastiCube table.
  • The image URL must be with http:// protocol.

To install the Embed Images add-on:

  1. Download the attachment and unzip the contents into your C:\Program Files\Sisense\PrismWeb\plugins\ folder. If the plugins folder doesn’t exist, just create it. After those files have been unzipped there, you may also have to restart the web server.
  2. Create a pivot or table widget. Make sure to add the field that contains the URLs of your images under the Rows section. At the upper-right corner of the widget editor, click the settings icon and look for the Image URL Column setting. This menu will list all the dimensions that have been added under Rows, so you can pick the one that has the URL to your images.
  3. Click Apply and refresh the page.
