Print Widget

The Print add-on adds the option to print a visualization to a widget’s menu.

After implementing the add-on, you can print your widget’s visualization by selecting Direct Print from the widget’s menu.

To install the Print add-on:

  • Download the attachment and unzip the contents into your C:\Program Files\Sisense\PrismWeb\plugins\ folder. If the plugins folder doesn’t exist, just create it. After those files have been unzipped there, you may also have to restart the web server. The Direct Print option is displayed in the right-click menu.
