Aggregated Tables

The Aggregated Table add-on extends the functionality of the Table widget by supporting aggregation across the whole table. This add-on allows you to sort the rows based on the lowest granularity across the entire table. This behavior is different from the regular table widget which groups the data, and supports sorting the rows inside each group, but not across the whole table.

To install the Aggregated tables widget:

  1. Download the attachment and unzip the contents into your C:\Program Files\Sisense\PrismWeb\plugins\ folder. If the plugins folder doesn’t exist, just create it.
  2. Create a new widget. The Aggregated table widget is displayed in the list of widget options.
  3. Select the Aggregated Table from the chart types menu.
  4. Click Create. The widget is added to the dashboard.
