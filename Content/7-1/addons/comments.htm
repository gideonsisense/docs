Comments

The Sisense Comments add-on enables you to add a comment box within the Sisense dashboard using the iFrame widget. Comment boxes provide a platform for annotation and social collaboration allowing viewers to discuss your visualizations.

This article describes how you can embed a simple 3rd party Comment Box tool by Bawks, but there are many other options available if other features are needed (log ins, more advanced moderator support, etc.).

Note: The comments in this example are hosted on a 3rd party site. If you require comments to be hosted internally, another script/tool may be used.

To set up the Comments add-on:

  1. Download the attachment and unzip the contents into your C:\Program Files\Sisense\PrismWeb\plugins\ folder. If the plugins folder doesn’t exist, just create it. After those files have been unzipped there, you may also have to restart the web server.
  2. Restart the IIS.
    If you installed IIS express when installing Sisense: Restart the service PrismWebExpress
    If you installed IIS when installing Sisense: Open the Command Prompt, type iisreset and click Enter.
  3. Set up a free HTML Comments Box Account. You can sign in with an existing Google account here to set up a free comments box. You may of course use another tool, but the setup may be slightly different.
  4. Copy the script given on the page once you have signed up. There are additional options available by clicking the options tab (these include several styling and metadata options), but this article describes the default options.
  5. Right-click this link, Comments.html, and select Save Link As to download the comments page and save it to the folder C:\Program Files\Sisense\PrismWeb.
  6. Using a text editor like Notepad ++, open the Comments.html file.
  7. Locate the following line:
    <!– INSERT SCRIPT HERE –>
    and replace this line with the script you copied from Step 4 and save the file.
  8. Open the Sisense dashboard you want to add the Comments box to. Create a new widget and click Advanced configuration.
  9. From the Widgets list, select the iFrame widget.
  10. On the right side of the page, in the IFRAME URL field, enter /Comments.html.
  11. Click Apply. An iFrame widget with the Comments add-on is displayed in your dashboard.
