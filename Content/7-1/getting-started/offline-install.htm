Installing Offline

The standard installation of requires internet access to download additional necessary components. To support machines behind firewalls, or without internet access, you need to download and install the full installation file. After you have installed , you need to provide an offline activation key.

The steps below describe how to install offline and activate it.

Prerequisites

Before installing offline, make sure you have the following installed:

  1. Microsoft .NET 4.6.1 Framework
  2. IIS – Internet information Services (Microsoft’s Web server), see the following instructions per your operating system
    Windows 7 or 8
    Windows Server 2008
    Windows Server 2012

Installing

To install offline:

  1. Download the latest full version of .
  2. Open the installation file and click Run. The Welcome to window is displayed.
  3. In the Welcome to window, click Activate.
  4. In the Activation window, click Behind a Firewall. The product ID is displayed.
  5. Copy the product ID.

  6. On an online machine, open http://www.sisense.com/my-account/ and log in with your credentials.
  7. In the My Account page, from the left menu, select GET KEY.
  8. Paste your product ID from the Sisense installation into the field and click GET KEY. Your Off License key is displayed.
  9. Copy the key and paste it into the key area in the Activation window.
  10. Click Install. is activated and you can begin to install Sisense.
  11. Click Custom Installation.
  12. Select Microsoft IIS and click Continue.

    The installation will start running and complete.
