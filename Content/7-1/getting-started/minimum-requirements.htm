Supported Web Browsers

The runs in the following HTML5 supported browsers:

  • Internet Explorer 10 and higher

  • Google Chrome

  • Firefox

Note: When embedding iFrames, supports Safari 10 and higher

The Web Application also works in mobile phone and tablet browsers that support HTML5. Click here to learn more about mobile compatibility.

Supported Operating Systems

and can be installed on the following 64-bit operating systems:

Recommendations

recommends that your server meets or exceeds the minimum requirements listed above. The actual requirements of your server may vary depending on the number of concurrent users, builds running in parallel, s hosted on a server, and additional factors specific to your server, for example, non- applications running on the same server.

For optimal performance, recommends:

  • No more than 40 s on a single server

  • No more than 4 concurrent builds at a time

  • No more than 150 concurrent users

For optimal performance, highly recommends that your servers have a processor that supports AVX (Advanced Vector Extensions), which is leveraged by for improved query performance and user concurrency.
